
# mod_dptools: digit_action_set_realm 

 

## 0\. About

Used with to [mod\_dptools: bind\_digit\_action](./6586414.mdx#0-about) to set the realm, for which bound digits will be applied against. 

See there for in-depth description, and examples.

## 1\. Usage

```xml
<action application="digit_action_set_realm" data="my_realm[,target]"/>
```

| Parameter | Description                                                   |
| --------- | ------------------------------------------------------------- |
| target    | The leg upon which to set the realm. Possible values:selfpeer |
